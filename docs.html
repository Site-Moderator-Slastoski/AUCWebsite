Documentation

This is the official documentation for the project Among Us: Classic, detailing setup, configuration, and development guidelines.

1. PC Installation Guide

1.1 Getting Started on PC

In order to continue this process you will need to join the Discord by clicking the Discord icon. All files in this guide all found via the Discord. You may create a ticket if you need support!

The files that are required are:

  • The Game Files (v2020.11.17)
  • The Region File

    • 1.2 Installing the Region File

    The Region File is a single file that tells your computer where our servers are and allows you to connect to them. You can download this via the Discord. In order to install this correctly, you need to put this file in a specific place. C:\Users\%username%\AppData\LocalLow\Innersloth\Among Us

    1.3 Downloading the Game Files

    This file is a simple .zip file and can be downloaded via the Discord. Once you have this .zip file, simply extract it and launch the game via "Among Us.exe"

    If you have any issues with this file, feel free to create a new ticket in the Among Us: Classic server. We'll help you right away!

    2. Android Installation Guide

    2.1 Getting Started on Android

    In order to continue this process you will need to join the Discord by clicking the Discord icon. All files in this guide all found via the Discord. You may create a ticket if you need support!

    The files you'll need to achieve a successful install are:

  • Among Us (v2020.11.17) APK
  • FolderV File Explorer
  • Shizuku (only on a few devices)
  • The Region File

    • 2.2 Installing the Game

    The first step is to install the Among Us (v2020.11.17) APK. Simply, open the APK file and click Install. If you encounter any errors, please resort to the Discord. Once installed, you will need to open the game in order for it to generate game files, which you will need to replace and modify in order to connect to our services.
